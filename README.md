# Ronaldo Santos | Dev No-Code

Este é o repositório do portfólio pessoal e página de apresentação de Ronaldo Santos, um Desenvolvedor No-Code especializado em **FlutterFlow** e **Supabase**. O projeto foi concebido para demonstrar habilidades, serviços prestados e projetos em destaque com um design moderno, escuro e altamente responsivo.

## 🚀 Sobre o Projeto

O portfólio foi desenvolvido utilizando **HTML5**, **CSS3** e **JavaScript**, sem depender de frameworks complexos. A sua estrutura foi desenhada para oferecer uma excelente experiência ao utilizador, com um visual elegante e futurista, inspirado em plataformas de tecnologia e inovação.

---

## 🛠 Tecnologias Utilizadas

O projeto foi construído com as seguintes tecnologias e ferramentas:

- **HTML5:** Estruturação semântica de todo o conteúdo.
- **CSS3:** Estilização avançada, incluindo:
  - **Variáveis CSS:** Para uma paleta de cores consistente (tons escuros, roxo, neon).
  - **Efeitos Visuais:** Efeito de *noise* (ruído) sobre o fundo e borrões (*blur*).
  - **Responsividade:** Layouts adaptáveis via CSS Grid e Flexbox para telemóveis e computadores.
  - **Animações:** Efeitos de transição e animações de entrada (*fade-up*).
- **JavaScript:** Lógica interativa para o carrossel, atualização de contador e renderização de pontos indicadores.
- **Fontes do Google:** Uso das famílias tipográficas *Syne* e *DM Sans*.

---

## 📂 Estrutura do Arquivo

O arquivo principal `portfolio-ronaldo.html` contém todo o site em uma única página (Single Page Application), organizado nas seguintes secções:

1. **Navegação (Nav):** Logótipo, links para as secções e botão de CTA.
2. **Hero Section:** Mensagem principal, descrição, botões de ação e cartão de perfil com badges.
3. **Estatísticas (Stats):** Indicadores de desempenho (Projetos entregues, tecnologias utilizadas).
4. **Sobre Mim:** Descrição detalhada e *cards* de competências (FlutterFlow, Supabase, Flutter, AI CLI).
5. **Serviços:** Grid com 6 serviços detalhados (Apps Mobile, Backend, Web Apps, etc.).
6. **Projetos:** Secção de portfólio destacando aplicações construídas.
7. **Contato:** Formulário de contacto, email, redes sociais e rodapé.

---

## 💻 Como Utilizar

Para visualizar o projeto localmente:

1. Clone o repositório ou faça o download do ficheiro `portfolio-ronaldo.html`.
2. Abra o ficheiro `portfolio-ronaldo.html` diretamente no seu browser (Google Chrome, Mozilla Firefox, Safari, etc.).
